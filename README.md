# 🛋 Мебельный Рай — интернет-магазин мебели

[![PHP](https://img.shields.io/badge/PHP-8.2+-777BB4?logo=php)](https://php.net)


**Мебельный Рай** — полноценный интернет-магазин для продажи корпусной и мягкой мебели.  
Проект состоит из трёх компонентов:

- **Веб-сайт** (PHP + MySQL) – клиентская часть для просмотра каталога, корзины и оформления заказов.

---

## 🚀 Технологический стек

| Компонент       | Технологии                                                                 |
|-----------------|----------------------------------------------------------------------------|
| **Backend**     | PHP 8.2                                                                    |
| **Frontend**    | HTML5, CSS3, Bootstrap 5, JavaScript (jQuery)                              |
| **База данных** | MySQL 8.0                                                                  |
| **Desktop-приложение** | C# .NET 8.0, Windows Forms, Entity Framework (подключение к той же БД) |
| **Версионирование** | Git, GitHub                                                          |
